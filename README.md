# Online Trading Website

## Description
Built an online trading website from  where users can user can buy and sell unlisted shares of a company. It is built using React as frontend and MongoDb as backend.
Added admin functionality through which admin can do all CRUD operations on the company. 

## Languages and Technology Used
- React
- Javascript
- Mongodb
- Html & Css
- Bootstrap
- React-Router
